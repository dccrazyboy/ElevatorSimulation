Instance Variable Details [hide private]

Status

当前电梯状态
  • 'Idle': Elevator处于空闲状态，等待请求。控制权为控制类。
  • 'Running': Elevator处于运行状态。控制权为自己。
  • 'Boarding': Elevator有顾客正在进入电梯。控制权为控制类。
  • 'Landing': Elevator有顾客正在离开电梯。控制权为控制类。
Type:
str

TimeRemain

表示某状态需要等待的时间

只在Status为'Running'、'Boarding'、'Landing'时有效

Type:
int
